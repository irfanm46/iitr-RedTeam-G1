# iitr-redteaming-g1

Collaborative repository for the Group 1 Red Team simulation project. This workspace tracks vulnerability analysis, log parsing, and risk remediation strategies.

## Project Structure
* `/data` - Workbook logs, OSINT findings, and raw analysis materials.
* `/docs` - Final milestone tracking, reports, and documentation.

## Team Breakdown & Tracks

###  OSINT & Reconnaissance
* Focus: Leaked credentials, data exposure, and public asset discovery.

###  Network Logs & API Analysis
* Focus: Network traffic logs, firewall rules, and API gateway enumeration.

###  Risk Assessment & Remediation (Irfan)
* Focus: CVSS scoring matrices, business impact analysis, and the 0–180 day strategic roadmap.
